Hier finden Sie eine kurze Beschreibung zur Bedienung dieser Site (die Navigation).

  1. Sie können im Titel Ihres Browser das Verzeichnis sehen, in dem Sie sich zur Zeit befinden. Es begint mit dem Namen der Site und zeigt dann die Menüstruktur an, in der Sie sich befinden, wobei jedes Untermenü getrennt ist mit einem Strich.
  2. Hier steht der eigentliche Titel der aktuellen Seite.
  3. An dieser Stelle werden Kurznachrichten angezeigt. Z. B. wenn die Seite fertig geladen wurde, erscheint dort kurz eine Meldung, aber auch wenn Sie mit der Maus über bestimmte Objekte gehen, wird dort kurz angezeigt, was für eine Funktion hinter dem entsprechenden Objekt steckt.
  4. Mit diesem Button kommen Sie auf Ihre zuvor betrachtete Seite auf dieser Site.
  5. Wenn Sie den Zurück-Button benutzt haben, können Sie so wieder zurückkommen.
  6. Hier kommen Sie in dem Menü auf das nächst höhere Untermenü, also die darüberliegende Verzeichnisstruktur.
  7. Mit diesem Button können Sie diese Site als Startseite für Ihren Browser einrichten. Es wäre mir eine Ehre, wenn Sie dies tun würden!
  8. So können Sie die Site in die Favoriten Ihres Browser aufnehmen, um sie leicht später wiederzufinden.
  9. Falls Sie nach bestimmten Begriffen auf dieser Site suchen wollen, können Sie das hier machen. Sie gelangen nach einem Klick auf diesen Button zu dem entsprechenden Menüpunkt und es wird dann eine Suche angezeigt.
  10. Wenn Sie die Seite verlassen wollen, müssen Sie einfach nur hier klicken, und das Fenster wird geschlossen.
  11. Hier wird das Menü angezeigt, das dank JavaScript dynamisch ist und eine sehr ähnliche Navigation wie im Windows-Explorer zulässt.
  12. Und hier sehen Sie dann die eigentliche Hauptseite mit dem Inhalt.

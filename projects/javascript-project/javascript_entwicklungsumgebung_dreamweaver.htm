Das Programm Dreamweaver ist etwas allgemeiner und hauptsächlich für die HTML-Programmierung gedacht. In diesem Bereich ist es aber mein absoluter Favorit. Es kann aber auch teilweise den JavaScript farblich hervorheben, sonst besitzt es aber keine besonderen Features für JavaScript. Dafür ist es für HTML umso besser. Der Code, den es erzeugt, ist sehr sauber und es ist sehr leicht zu bedienen.

Hier ein Screenshot:

Da dieses Programm von Macromedia ist, ähnelt es nur bedingt anderen Entwicklungsumgebung. Wie man sieht, benutzt es noch SDI, aber in der neuen Version soll jetzt auch alles in einer MDI-Form angezeigt werden.

Links befinden sich die Standard-Steuerelemente für HTML. Unten in dem Fenster kann man jetzt die Eigenschaften von dem entsprechenden Objekt festlegen. In dem oberen rechten Fenster kann man ziemlich schnell und leicht zwischen den definierten CSS-Stylen hin und her schalten. Das Fenster darunter zeigt eine kleine Hilfe zum Nachschlagen an. In dem größten Fenster in der Mitte kann man dann den eigentlichen Code eingeben. Man kann dort aber auch zu der WYSIWYG-Ansicht wechseln, bei der man dann direkt die HTML-Objekte platzieren kann und einen Text dazu schreiben kann. Das ganze sollte dann genauso, wie man es dort eingibt und es dort zu sehen ist, auch im Browser aussehen, was ja auch der Name der Ansicht sagt (WYSIWYG = what you see is what you get).

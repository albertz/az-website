Wie in allen Programmiersprachen fange ich hier erstmal mit dem "Hallo Welt"-Beispiel an.

Sie müssen als erstes den Skript-Bereich in deiner HTML-Seite definieren (wenn Sie keinen Editor benutzen, der das automatisch macht). Dieser Skript-Bereich ist einfach ein HTML-Tag, der den Code umschließt. Damit dieser Code ältere Browsers nicht durcheinander bringt, sollte man aus dem Code dazwischen einen HTML-Kommentar machen. Das ganze sollte dann in etwa so sein:
<html>
<head>
<title>Test</title>
<script language=JavaScript>
<!--
JavaScript-Code...
//-->
</script>
</head>
<body>
Hier jetzt die eigentliche Seite...
</body>
</html>
Ereignisse müssen jetzt bei den entsprechenden HTML-Tags definiert werden. Wenn man z. B. einen Code beim Betreten der Seite ausführen will, dann ist das onload-Ereignis von body das richtige Ereignis. Man müsste den Body-Tag dann so umändern:
<body onload="JavaScript-Code">

Kommen wir zu unserem einfachen "Hallo Welt"-Beispiel zurück. Unser Ziel ist es, dass beim Aufrufen der Seite diese Meldung kommt. Für Meldungen gibt es den JavaScript-Befehl alert. Mann muss also in das onload-Ereignis folgendes schreiben:
alert("Hallo Welt");

Sie können sich diese Beispiel jetzt betrachten, indem Sie im Menü auf den entsprechenden Eintrag klicken. Das Beispiel dort ist noch etwas erweitert um ein weiteres Ereignis, bei dem aber auch nur ein alert ausgeführt wird.

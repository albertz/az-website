Willkommen auf dieser Site!

Diese Seite ist für alle, die vor haben, Programmieren zu lernen und auch erfolgreich anzuwenden, aber auch für erfahrene Programmierer zum Nachschlagen. Sie finden hier hauptsächlich Beispiele, Informationen, Erklärungen und Tipps zu Visual Basic und JavaScript.

Diese beiden Programmiersprachen Visual Basic und JavaScript sind beide extrem leicht und schnell zu erlernen, wobei Visual Basic für die professionelle Programmierung von Windows-Anwendungen gedacht ist und JavaScript eher eine so genannte Skriptsprache ist, was man auch schon im Namen erkennen kann, die für das Erstellen von dynamischen HTML-Seiten gedacht ist.

Bei JavaScript muss man noch dazu sagen, dass es hier 2 verschiedene Möglichkeiten gibt, diese Skripte anzuwenden. Sie können entweder direkt auf dem Webserver ausgeführt werden und liefern dann die fertige HTML-Seite an den Clienten oder sie werden im Browser des Clienten ausgeführt, was allerdings einen großen Nachteil hat, denn die verschiedenen Browser (Internet Explorer, Netscape, ...) führen diesen Code leider etwas unterschiedlich aus. Allerdings ist die erste Möglichkeit doch noch etwas komplizierter und verlangt einiges an Wissen, so dass ich hier nur auf den normalen client-seitigen Code eingehe.

Zur Demonstration von JavaScript wurden diese Seiten übrigens mit dieser Skriptsprache optimiert. Diese Menüführung wäre mit reinem HTML nicht möglich. Wenn Sie bereits etwas Erfahrungen mit JavaScript gesammelt haben, können Sie selbstverständlich den Code dieser Site betrachten und auch frei kopieren, wenn sie in einem kurzen Kommentar des Codes angeben, dass der Code von mir stammt.

Viel Spaß auf dieser Site und viel Erfolg bei Ihrer zukünftigen Programmierer-Karriere!

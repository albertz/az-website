Da JavaScript keine richtige Programmiersprache ist, sondern nur eine Skriptsprache, gibt es leider hier keine einheitliche Entwicklungsumgebung, sondern hier haben mehrere Firmen Ihre eigenen Software dafür programmiert. Im Prinzip kann man JavaScript auch mit dem Editor programmieren, aber so eine Entwicklungsumgebung erleichtert das Programmieren häufig bedeutend. Vor allem die farbige Hervorhebung des Codes macht das Programmieren deutlich einfacher, da der Code so viel übersichtlicher wirkt.
